# aplikasipenilaiansikapsiswa
Aplikasi Penilaian Sikap Siswa adalah sebuah sistem berbasis website yang dirancang untuk membantu guru dalam melakukan evaluasi sikap siswa secara digital, terstruktur, dan efisien. Aplikasi ini memungkinkan guru untuk memasukkan data siswa berupa nama dan jenis kelamin, kemudian memberikan penilaian melalui beberapa pertanyaan sikap dengan skala nilai 1 sampai 100.
Setelah nilai diinput, sistem akan secara otomatis menghitung rata-rata dan mengelompokkan hasil ke dalam kategori penilaian, yaitu Kurang Baik, Cukup Baik, Baik, dan Sangat Baik. Semua data yang dimasukkan akan tersimpan dan ditampilkan dalam bentuk rekap sehingga memudahkan guru dalam memantau perkembangan sikap siswa secara berkala.
Aplikasi ini memiliki tampilan sederhana namun modern, mudah digunakan, serta dapat diakses melalui perangkat komputer maupun smartphone sehingga mendukung proses penilaian yang lebih praktis dan profesional.
